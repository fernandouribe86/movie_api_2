Add a movie to Favorites List

Request URL HTTP Method Request Body Data Format Response Body Data Format
Allow an Existing User to add a movie to their Favorites list. /users/:Username/movies/:MovieID POST A push of the selected MovieID from the movies database to the their list of favorites. User receives a confirmation that the task was completed successfully/unsuccessfully.

Remove a movie to Favorites List

Request URL HTTP Method Request Body Data Format Response Body Data Format
Allow an Existing User to remove a movie to their Favorites list. /users/:Username/movies/:MovieID PUT A string containing the user's username and another string containing the MovieID. User receives a confirmation that the task was completed successfully/unsuccessfully.
