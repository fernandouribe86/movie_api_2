db.users

GET
/users Description: Returns a list of users.
Parameters Request Body Data Format Response Body Data Format
None None
(Code 201) A JSON object holding an array containing data about users, structured like:

[{
 _id: ObjectId("56789"),
 Username: 'AppleBottomJeans',
 Email: 'BootsWthe4@domain.com'
 Password: 'Wh0leCLu8L00k!ng'
 Birthday: '2001-01-01'
 Favorites: [ObjectID("5465465"), ObjectID("546413"]
 },
{
 _id: ObjectId("132456"),
 Username: 'SheHitDFloor',
 Email: 'NextThingYouKnow@domain.com',
 Password: 'Sh@wtyG0tL0w!',
 Birthday: '2011-11-11',
 Favorites: [ObjectID("123414"), ObjectID("456151"]
 }]
GET
/users/:Username Description: Returns a single User by Username.
Parameters Request Body Data Format Response Body Data Format
A user's Username ("Username") A string containing the requested user's Username (i.e. "SheHitDFloor").
(Code 201) A JSON object holding data about a single user, structured like:

{
 _id: ObjectId("132456"),
 Username: 'SheHitDFloor',
 Email: 'NextThingYouKnow@domain.com',
 Password: 'Sh@wtyG0tL0w!',
 Birthday: '2011-11-11',
 Favorites: [ObjectID("123414"), ObjectID("456151"]
 }
POST
/users Description: Create a new user.
Parameters Request Body Data Format Response Body Data Format
A user's desired Username ("Username") to verify if it is available.
A JSON object holding data about a single new user, structured like:

{
 Username: 'BaggySweatPants', *required
 Email: 'ReeboksWDStraps@domain.com', *required
 Password: '2TheWind0w!', *required
 Birthday: '2004-06-08' *required
 }
(Code 201) A confirmation that the user was successfully created.
PUT
/users/:Username Description: Allow a user to update their information.
Parameters Request Body Data Format Response Body Data Format
The user's username ("Username").
A JSON object holding data about a single new user, structured like:

{
 Username: 'BaggySweatPants456', *required
 Email: 'ReeboksWDStraps@domain.com', *required
 Password: '2TheWind0w456!', *required
 Birthday: '2004-06-11' *required
 }
A confirmation that the user's information was updated successfully.
POST
/users/:Username/movies/:MovieID Description: Allow a user to add a movie to their Favorites.
Parameters Request Body Data Format Response Body Data Format
A user's Username and a user's desired Movie by ID. A string containing a user's Username and a separate string containing the movie's ID. A confirmation that the movie was successfully added to the user's account (updatedUser).
PUT
/users/:Username Description: Allow a user to update their information.
Parameters Request Body Data Format Response Body Data Format
The user's username ("Username").
A JSON object holding data about a single new user, structured like:

{
 Username: 'BaggySweatPants456', *required
 Email: 'ReeboksWDStraps@domain.com', *required
 Password: '2TheWind0w456!', *required
 Birthday: '2004-06-11' *required
 }
A confirmation that the user's information was updated successfully.
DELETE
/users/:Username/movies/:MovieID Description: Allow a user to remove a movie to their Favorites.
Parameters Request Body Data Format Response Body Data Format
A user's Username and a user's desired Movie by ID. A string containing a user's Username and a separate string containing the movie's ID. A confirmation that the movie was successfully removed to the user's account (updatedUser).
DELETE
/users/:Username Description: Allow a user to delete their account by Username.
Parameters Request Body Data Format Response Body Data Format
A user's Username. A string containing a user's Username. (Code 200) A confirmation that the user's information was successfully deleted.
(Code 400) A user with that information was not found and therefore not deleted.
