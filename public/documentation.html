URL Endpoints

Movie List

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of my favorite movies upon request /movies/ GET None
A JSON object holding data about movies, structured like:

{
 ID: "M20",
 title: 'INSIDE',
 movieYear: 2021,
 directors: [
  {
   directorID: "D20",
   directorName: "Bo Burnham",
   directorData: {
    directorBio: "A short bio.",
    directorYOB: 1990,
    directorYOD: 2030,
    directorPhoto: "assets/bo_burnham.jpg",
    }
   }],
 movieDescription: "A brief description",
 movieGenres: ["Comedy", "Drama", "Music"],
 movieImage: "assets/bo.jpg",
 featured: true
}

ID

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a movie ID on search /movies/:ID GET None An alpha-numberic database code associated with the movie.

Title

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a movie title on search /movies/:title GET None A string containing the movie's title.

Movie Release Year

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a movie's year of release on search /movies/:title/movieYear GET None A number containing the year the movie was released.

Movie Description

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a movie's description. /movies/:title/movieDescription GET None A string containing the movie's description.

Movie Main Image

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a movie's theatrical poster image. /movies/:title/movieImage GET None A string containing the URL or relative path containing the movie's theatrical poster in .jpg (or .jpeg) format.

Featured

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a movie's featured tag. /movies/:title/featured GET None A boolean result on whether the movie is featured (true) or not (false).

Director(s)

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get movie's director(s). /movies/:directors GET None
A JSON object holding data about the movie's director(s) in an array, structured like:

{
 directors: [
  {
   directorID: "D20",
   directorName: "Bo Burnham",
   directorData: {
    directorBio: "A short bio.",
    directorYOB: 1990,
    directorYOD: 2030,
    directorPhoto: "assets/bo_burnham.jpg",
    }
   }],
}

Genres

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a movie's theatrical poster image. /genres GET None
An array containing JSON objects holding data about potential genres structured like:

{
 genre: "Animation.",
 genreDescription: "A brief genre description."
 }

Genre Names

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a genre. /genres/:genre GET None A string containing the genre name.

Genre Names

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a genre. /genres/:genreDescription GET None A string containing the description of the requested genre.
